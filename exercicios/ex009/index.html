Listas Ordenadas

A tag <ol> ordena a lista e pode ter uma parâmetro para mudar a contagem.

A tag <li> é cada item de lista e o fechamento da tag é opcional.

  1. Acordar
  2. Escovar os dentes
  3. Tomar café da manhã
  4. Ligar o computador
  5. Estudar
  6. Almoçar
  7. Assistir uma série
  8. Jogar uma jogo
  9. Jantar
  10. Dormir

Listas não Ordenadas

A tag <ul> lista não ordenada e pode ter uma parâmetro para mudar a marcação.

Listas de Definições

A tag <dl> lista de definições como se fosse um dicionário.

A tag <dt> Termo. A tag <dd> Descrição.

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
